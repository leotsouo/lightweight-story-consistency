# 輕量智敘 Product Spec

## 產品名稱
輕量智敘：結合邊緣部署與輕量化 LLM 的長篇敘事一致性檢核系統

## 產品目標
打造一個 Web App 雛型，讓使用者可以上傳長篇文本，查看系統對角色、事件、物件、地點與世界觀規則的一致性檢核結果，並以視覺化方式呈現衝突報告與知識圖譜。

## 目標使用者
- 遊戲敘事設計師
- 影視編劇與企劃團隊
- 出版編輯
- 原創 IP 團隊

## 核心問題
長篇文本在多次修訂後，容易發生：
- 人物關係混亂
- 事件前後矛盾
- 設定遺漏
- 世界觀規則不一致

## 核心功能
1. 邏輯衝突偵測
2. 角色一致性追蹤
3. 知識圖譜輔助檢核
4. 多代理人協作分析
5. 本地端部署導向的操作介面

## 雛型頁面需求
### 1. Dashboard 首頁
- 專案總覽
- 最近檢核結果
- 衝突數量摘要
- 角色數 / 事件數 / 規則數
- 最近版本比較摘要

### 2. 專案/文本上傳頁
- 上傳劇本/小說/企劃文本
- 顯示版本資訊
- 顯示檔案狀態與分析進度

### 3. 檢核結果頁
- 衝突總數
- 衝突分類卡片（角色/時間/物件/世界觀）
- 衝突列表
- 可點開看前後文

### 4. 角色一致性頁
- 角色清單
- 角色設定摘要
- 各章節行為/能力/性格變化
- 可能漂移警示

### 5. 知識圖譜頁
- 顯示角色、物件、事件、地點關聯
- 可以點擊節點看詳細資訊

### 6. 報告頁
- 輸出衝突報告
- 顯示建議修正方向
- 可模擬 PDF 匯出按鈕

## UI 風格
- 現代 SaaS 風格
- 深色或淺色皆可，優先做質感深色
- 乾淨、專業、帶 AI 產品感
- 以產品展示 demo 為目標，不必真的串後端模型

## 技術偏好
- Next.js
- Tailwind CSS
- 可用 shadcn/ui
- 使用 mock data
- 前端 prototype 即可
