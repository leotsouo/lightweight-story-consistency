## 輕量智敘：長篇敘事一致性檢核系統（前端 Prototype）

這是一個以 **Next.js + Tailwind CSS** 打造的前端雛型專案，用於展示「輕量智敘：結合邊緣部署與輕量化 LLM 的長篇敘事一致性檢核系統」的產品概念。  
整體以 **Demo 展示 / 產品 Prototype** 為目標，尚未串接真實後端與模型，所有數據皆為 **mock data**。

---

### 功能與頁面總覽

- **Dashboard 首頁**
  - 專案總覽、最近檢核結果
  - 衝突數量摘要（角色 / 事件 / 規則等）
  - 角色數 / 事件數 / 規則數
  - 最近版本比較摘要

- **專案／文本上傳頁**
  - 上傳劇本／小說／企劃文本
  - 版本資訊顯示
  - 檔案狀態與分析進度（以 mock 狀態條呈現）

- **檢核結果頁**
  - 整體衝突總數
  - 依類別的衝突卡片（角色／時間／物件／世界觀等）
  - 衝突列表，可點擊展開前後文片段

- **角色一致性頁**
  - 角色清單與角色設定摘要
  - 各章節的行為／能力／性格變化
  - 可能「角色漂移」的警示標示

- **知識圖譜頁**
  - 以圖像化方式呈現角色、物件、事件、地點的關聯
  - 可點擊節點檢視詳細資訊（mock 視窗／側欄）

- **報告頁**
  - 顯示衝突彙整與建議修正方向
  - 提供模擬的「匯出 PDF 報告」按鈕（無實際匯出邏輯）

---

### 技術棧

- **框架**：Next.js（App Router）
- **樣式**：Tailwind CSS
- **UI 元件**：可搭配 shadcn/ui（依實作情況）
- **資料來源**：前端內建 mock data，無實際 LLM 推論流程

---

### 本機開發與執行方式

1. 安裝依賴套件：

```bash
npm install
```

2. 啟動開發伺服器：

```bash
npm run dev
```

3. 開啟瀏覽器並前往：

```text
http://localhost:3000
```

如有修改程式碼，頁面會自動熱重載。

---

### 專案結構（摘要）

以下僅列出與產品 Prototype 相關的主要目錄（實際結構可依最終程式為準）：

- `app/`：Next.js App Router 頁面與 layout
- `components/`：可重用的 UI 元件（卡片、表格、圖譜視覺化等）
- `lib/` 或 `mock/`：mock data 與前端資料模型
- `docs/`：產品說明與規格文件（例如 `product_spec.md`）

---

### 產品定位說明

- 這個專案主要用於 **展示敘事一致性檢核平台的操作介面與資訊架構**。
- 重點在於 UI 完整度、互動流程與資訊呈現，而非演算法或模型實作。
- 若要接上真實的 LLM 或邊緣部署環境，可在此 Prototype 基礎上替換 mock data 呼叫流程，並串接實際的後端 API。
